## 项目简介

这个项目是一个简易易用的汽车销售平台，提供用户能够浏览、评论和购买汽车的平台。平台采用最新的技术和框架，以确保性能和可靠性。

## 主要功能

- 用户系统：登录、注册、退出功能
- 车辆信息：查看、详情、评论、购物车、购买
- 地址管理：新增、查看、修改、删除
- 购车记录：查看购买记录
- 个人中心：修改用户信息
- 分类管理：查看、新增、修改、删除


## 技术架构

- 前端：React + TypeScript
- 后端：Node + Express + MongoDB + Redis
- 数据库：MongoDB，Redis


## 使用说明

1. 代码包已包含所有所需的依赖包和配置。
2. 启动项目：
    - 运行 `npm install` 以安装依赖包。
    - 运行 `npm start` 以启动应用程序。
